# ANTI-BULLET

8ビット風レトログラフィックとピコピコBGMで楽しむ、極限の弾幕避けシューティングアクションゲーム！

## 概要

**ANTI-BULLET** は、Vanilla JavaScriptとCanvas 2D APIで実装されたシューティングゲームです。プレイヤーは自機を操作して、画面上から降ってくる敵の弾幕を避け、生き残った時間に応じてスコアを獲得します。

## 特徴

### ゲームシステム
- **3つの難易度**: EASY (5ライフ, 0.5倍スコア) / NORMAL (3ライフ, 1.0倍スコア) / HELL (1ライフ, 2.0倍スコア)
- **3つの特殊能力**: 
  - CLASSIC: 特殊能力なし (1.5倍スコア)
  - TIME SLOW: スペースキーで時間をスローダウン (SPACE)
  - SHIELD: スペースキーでバリアを展開 (SPACE)

### 敵タイプ
1. **直進敵 (Basic)**: シンプルな敵、プレイヤーに向かって3方向扇状に弾を発射
2. **追尾敵 (Tracking)**: プレイヤーを追いながら4方向全周に弾を放出
3. **螺旋敵 (Spiral)**: ゆっくり降りながら回転する螺旋パターンで弾を発射 ⭐ NEW
4. **ボス**: 生存時間60秒ごとに出現、複雑な弾幕パターンを展開

### ビジュアル
- **レトロピクセルアート**: 8x8グリッドで描かれたキャラクター
- **CRTモニター風エフェクト**: スキャンライン、ビネット、走査線
- **パーティクル効果**: 敵撃破時の爆発、プレイヤーのエンジン火花
- **ネオンカラー**: シアン、マゼンタ、黄色のレトログラフィック配色

### サウンド
- **ピコピコBGM**: Web Audio APIで生成される8ビット風BGM
- **ダイナミック音**: タイムスロー中にテンポが変わるBGM
- **効果音**: 射撃音、被弾音、スローSFX、シールド音

## 操作方法

| キー | 機能 |
|------|------|
| **↑/W** | 上移動 |
| **↓/S** | 下移動 |
| **←/A** | 左移動 |
| **→/D** | 右移動 |
| **SPACE** | 特殊能力発動（TIME SLOWまたはSHIELD） |
| **ENTER** | メニュー確定 |
| **ESC** | タイトルに戻る |

## ゲームシステム詳細

### スコアシステム
- **生き残りボーナス**: 毎フレーム自動的にスコアが増加
- **難易度スコア倍率**: EASY (0.5x) / NORMAL (1.0x) / HELL (2.0x)
- **特殊能力スコア倍率**: CLASSIC (1.5x) / TIME SLOW (1.0x) / SHIELD (1.0x)
- **ハイスコア**: 難易度・能力の組み合わせごとに localStorage に保存

### 当たり判定
- **プレイヤーの当たり判定**: 半径 1.0 ピクセル（非常に厳しい判定）
- **弾のサイズ**: 通常弾 4px、ボス弾 6px

### 敵出現頻度
- **最初**: 敵が少なく、時間とともに激化
- **難易度**: HELL では敵が最も多く、弾も高速化
- **ボス出現**: 生存時間 60 秒ごとに出現

### 時間スロー (TIME SLOW)
- **効果**: ゲーム内の敵と弾の更新速度が 30% に低下
- **BGM**: テンポが変わるダイナミック反応
- **ゲージ**: 消費しながら使用でき、使用しないと回復

### シールド (SHIELD)
- **効果**: 展開中（1.5秒）、周囲 24px 内の弾を破壊
- **クールダウン**: 展開後 8 秒のクールダウン
- **ビジュアル**: シアン色の鼓動する円で表示

## 技術スタック

- **言語**: Vanilla JavaScript (ES6+)
- **レンダリング**: Canvas 2D API
- **オーディオ**: Web Audio API
- **ピクセルアート**: 2D配列で定義されたドット絵データ
- **データ永続化**: localStorage（ハイスコア保存）
